## What's New

- The Dictionary now has a faster split-button add flow, native right-click edit and delete actions, and keyboard selection and deletion for entries.
- Custom vocabulary now carries through streaming post-stop enhancement, preserving preferred spelling and capitalization just like batch dictation.
- AI Enhancement now separates built-in and custom presets, and built-in prompts can be copied into an editable custom preset.

## Improvements

- Dashboard statistics now measure direct voice dictation only, so imported media and other transcription sources no longer inflate dictation totals.
- Streaming finalization now follows the configured transcription-enhancement assignment instead of a separate streaming-only toggle.

## Bug Fixes

- Pindrop can now repair older SwiftData stores that are missing newer migration columns, preserving transcription history instead of failing during launch.
