## What's New

- The floating Orb can be dragged again, and its right-click quick actions — including temporarily hiding it — are restored.

## Bug Fixes

- Fixed an update-related database failure that could prevent Pindrop from launching after SwiftData store metadata had been repaired with an incompatible schema. Pindrop now restores the correct versioned metadata while preserving existing transcriptions.
- Fixed the “Failed to save speaker identities” error that could appear when opening Settings → Dictation. The production store now uses the explicit current schema and is validated before the app keeps the container.
- Floating-indicator mouse events now route through their panels consistently, restoring drag and context-menu interactions without breaking normal clicks.
